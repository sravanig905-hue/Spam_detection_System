# Spam Detection System

## Project Overview

The Spam Detection System is a Machine Learning project designed to classify messages as Spam or Ham (Not Spam). The model analyzes text messages, extracts meaningful features, and predicts whether a message is spam using Natural Language Processing (NLP) techniques and machine learning algorithms.

## Objectives

* Detect spam messages automatically.
* Improve message filtering efficiency.
* Apply NLP techniques for text preprocessing.
* Build and evaluate a machine learning classification model.

## Dataset

The dataset contains text messages labeled as:

* Spam
* Ham (Not Spam)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

### 1. Data Collection

* Imported the spam message dataset.

### 2. Data Preprocessing

* Removed special characters and punctuation.
* Converted text to lowercase.
* Removed stop words.
* Tokenization and text cleaning.

### 3. Feature Extraction

* Count Vectorization
* TF-IDF Vectorization

### 4. Model Building

* Trained machine learning classification models.
* Evaluated model performance using accuracy metrics.

### 5. Prediction

* Classified incoming messages as Spam or Ham.

## Results

The model successfully identifies spam messages and helps automate message filtering, improving communication efficiency and reducing unwanted content.
